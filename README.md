# 🚀 DevTracker Pro

DevTracker Pro is a full-stack SaaS application designed to help developers manage and track their projects, tasks, and daily progress in one centralized dashboard. 

Built with a focus on clean architecture and modern deployment practices, it demonstrates a complete vertical slice of a production-ready application, from relational database design to a responsive user interface.

## 🛠 Tech Stack

*   **Frontend:** React (Vite), TypeScript, Tailwind CSS
*   **Backend:** Node.js, Express, TypeScript
*   **Database:** MySQL, Prisma ORM
*   **Security:** JSON Web Tokens (JWT), Bcrypt

## 📂 Project Structure

This repository is divided into two main environments. Please refer to their individual READMEs for specific setup instructions.

*   [`/backend`](./backend/README.md) - The Express API and MySQL database configuration.
*   [`/frontend`](./frontend/README.md) - The React user interface.

## 🚀 Quick Start

To run this project locally, you will need two separate terminal windows.

**1. Start the Backend API**
\`\`\`bash
cd backend
npm install
npm run dev
\`\`\`

**2. Start the Frontend UI**
\`\`\`bash
cd frontend
npm install
npm run dev
\`\`\`
